Turn MIDI

Drop in a MIDI file and this page will parse the notes, reuse repeated pitches as shared blocks, generate animated note triggers, and export a dark GRAB-style JSON level. Playback is synced to the MIDI length, and repeated notes retrigger the same pitch sound cleanly.

PreviewPiano-roll timeline with a moving playhead and active note glow.
ReuseRepeated pitches share the same block slot to keep exports tidy.
TriggerEvery note onset can retrigger sound and animate the matching block.
ExportDownload a GRAB-style JSON structure with levelNodes and triggers.
GRAB levels are stored as .level files in protobuf form, and this page focuses on a browser-side generator that builds a clean JSON representation you can adapt to your own workflow. ([github.com](https://github.com/Slin/GRAB-Level-Format?utm_source=chatgpt.com))

Notes are parsed with @tonejs/midi and playback uses Tone.js. ([github.com](https://github.com/Tonejs/Midi?utm_source=chatgpt.com))
