midi · grab converter

Turn MIDI
into GRAB.

Upload a MIDI file and this page turns it into a GRAB-style level JSON with reusable sound nodes, trigger nodes, and animated colored blocks that move into each trigger on time. Repeated pitches reuse the same sound node so the export stays compact.

SoundOne sound node per pitch, with MIDI-to-frequency mapping and retriggering.
TriggerEach note instance gets a trigger node wired to a block and sound target.
MotionColored blocks animate forward to hit the trigger at the note time.
ExportOutputs a JSON layout ready to inspect in GRAB Tools editor.
GRAB Tools’ editor is a JSON/3D level editor with trigger and animation viewing plus JSON export, so matching the node graph in JSON is the right direction for import workflows. ([grab-tools.live](https://grab-tools.live/editor?utm_source=chatgpt.com))

The browser uses @tonejs/midi for parsing and Tone.js for playback preview. ([github.com](https://github.com/twhlynch/grab-tools.live/blob/main/editor.html?utm_source=chatgpt.com))
